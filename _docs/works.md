### Foosball Referee

网页版桌式足球裁判应用，无需安装直接使用，支持裁判需要常见功能。Android 部分机型可支持震动提醒。

- 使用地址 <https://crispgm.github.io/foosball-referee/>
- GitHub: <https://github.com/crispgm/foosball-referee>

> This web app is highly inspired by FoosRef app.
